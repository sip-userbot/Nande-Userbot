### Tutorial Deploy di Termux
-  `pkg install git`
-  `git clone https://github.com/sip-userbot/Nande-Userbot`
-  `cd Nande-Userbot`
-  `bash termux.sh`

-  `cd Nande-Userbot`
-  `bash vps.sh`

</details>
